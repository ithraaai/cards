# 🕋 إثراء التجربة | نظام إدارة العمليات

نظام إدارة عمليات ومتابعة تنفيذ لشركة إثراء التجربة لخدمات حجاج الداخل.

## ✨ المميزات

- 9 فرق عمل مع معاييرها الكاملة (مدمجة من النظام الأصلي)
- فريق إدارة النفايات الكامل مع التقارير اليومية
- فريق النظافة مدمج ضمن الفرق العادية
- مقياس تقييم متعدد: نعم/لا، تقييم 1-5، رقمي، نصي
- حقل ملاحظات ذكي مع إلزامية مشروطة
- خيار "غير منطبق" لكل معيار
- قفل الأيام المستقبلية
- حفظ تلقائي
- 4 KPIs ديناميكية
- متجاوب بالكامل مع الجوال

## 🚀 التشغيل المحلي

```bash
npm install
npm run dev
```

افتح `http://localhost:5173` في المتصفح.

## 🏗 البناء للإنتاج

```bash
npm run build
```

سيُنشأ مجلد `dist/` يحتوي الموقع الجاهز للنشر.

## 🌐 النشر على Cloudflare Pages

1. ادفع الكود إلى GitHub
2. اربط مستودعك في Cloudflare Pages
3. استخدم الإعدادات التالية:
   - Build command: `npm run build`
   - Build output directory: `dist`
   - Node version: 20
